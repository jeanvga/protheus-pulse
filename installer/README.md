# Instalador

O instalador reproduzível em **Inno Setup** será implementado na Fase 5. Ele instalará binários em `C:\Program Files\Protheus Pulse`, manterá dados em `C:\ProgramData\ProtheusPulse`, registrará o serviço com início automático atrasado e preservará dados durante atualização.

Nenhum instalador incompleto é distribuído nesta fase.
